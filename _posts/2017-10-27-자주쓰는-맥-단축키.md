# 맥 단축키
- 부분선택 :  option + shift + 방향키
- 한영전환 : control + space
- 선택스크린샷: shift + command + 4
- 창스크린샷: shift + command + 4 + space
- 단어삭제: option + delete
- 전체화면: command + control + f
- 강제종료: command + option + esc
- 바탕화면보기: F11

# 크롬 단축키
- 새로고침 : command + R
- 구글검색 : command + L
- 페이지 내 검색 : command + F
- 개발자도구 : alt + command + I
