# 핸디즈 프론트엔드 인턴 (2018.01~2018.02)

2018 동계방학에 2달간 프론트엔드 개발자로 인턴을 하게 되었습니다.
핸디즈는 에어비앤비, 게스트하우스, 쉐어 하우스/오피스 등 모든 공간의 토탈 케어 솔루션을 제공합니다.
기획자 1명, 디자이너 1명, 백엔드개발자 분들과 협업하여 프로젝트를 진행했습니다.

## 핸디즈 메인 랜딩 페이지 개발

* 플로우 차트 설계
* 핸디즈 메인 페이지 개발
* 코드 리뷰
* 구글 애널리틱스를 이용하여 실시간 이벤트 확인
