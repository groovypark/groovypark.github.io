## 시간복잡도

재귀함수 return 함수(n-1)+(n-2)일 때 : O(2의 n제곱)

* log(N) : 효율이 좋은 검색 알고리즘. ex)이진탐색
* N : 입력자료 각각에 일정량이 할당되는 경우
* NlogN : 효율 좋은 정렬 알고리즘 ex)quick sorting, heap sorting
* N제곱 : 이중 루프
* N세제곱 : 삼중 루프

---

* 피보나치
  * 재귀호출 : 2의 N제곱, 스택오버플로우 문제 야기
  * 동적계획법(DP) : N
* 이진탐색(binary search) : log(N)
* 다익스트라 알고리즘 : V의 제곱
